vimconf
=======

##準備
###gitの設定
* 名前
* 改行コード
* プロキシ

##初期化
###Windows

* vimrcのあるフォルダでgit clone
* _vimrc, _gvimrcがある場合は削除する
* init.batを管理者として実行

###Mac

* ホームディレクトリに.vimディレクトリを作成
* .vimディレクトリでgit clone

##改行コードが問題になった時

###^Mが問題の場合
* vim -b （ファイル名）でファイルを開く。（^Mが表示される）
* :%s/^M//gで全て置換する
* ^Mは<C−v><C−m>で入力する
